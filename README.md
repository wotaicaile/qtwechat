# qtwechat
用pyqt5模仿微信界面（网络编程大作业）  
网络编程、多线程、数据库、ui
介绍：  
·该系统基于PYQT5、python3.9、mysql开发，界面参考微信，通过socket进行网络聊天通信。  
·	具有验证用户登陆、聊天消息发送、动态显示聊天气泡等功能。  
环境依赖：  
·客户端打包后不依赖环境，跨平台可用，如需运行源代码，需安装PYQT5、python3.9、mysql8.0  
·服务端为python脚本，需安装python3+，mysql8.0数据库管理系统使用。    
使用说明：  
根据chatInfo.txt创建数据库  
运行chatServer.py  
运行loginrun.py
